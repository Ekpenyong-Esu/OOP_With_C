# Real-Time Static Priority Project

This project demonstrates a real-time system using static priority scheduling with FreeRTOS. It includes modules for motor control, data management, display, and position sensing.

## Features

- **Real-Time Scheduling**: Tasks are scheduled using FreeRTOS with static priorities.
- **Modular Design**: Separate modules for motor control, data handling, display, and position sensing.
- **FreeRTOS Integration**: Utilizes FreeRTOS APIs for task management and scheduling.
- **Code Documentation**: Doxygen is used for generating documentation.

## Modules

### MotorController
Manages motor movements and ensures real-time control.

### MotorData
Handles data storage and retrieval for motor positions.

### MotorDisplay
Displays motor data periodically.

### MotorPositionSensor
Simulates and provides motor position data.

## Software Requirements

- CMake 3.21+
- FreeRTOS
- GCC or Clang
- Doxygen (for documentation)

## Building and Running

1. Clone the repository:
   ```shell
   git clone <repository-url>
   cd realtime_staticPriority
   ```

2. Build the project:
   ```shell
   mkdir build && cd build
   cmake ..
   make
   ```

3. Run the application:
   ```shell
   ./app/main
   ```

4. Run unit tests:
   ```shell
   ./tests/unit_tests
   ```

5. Generate documentation:
   ```shell
   make docs
   ```
